# Support

## Getting help

For usage questions and troubleshooting:

- Open a GitHub issue using the templates in `.github/ISSUE_TEMPLATE/`
- Include reproducible steps, expected behavior, and relevant logs

## Security issues

Do **not** file public issues for security vulnerabilities.
Follow the private reporting process in `SECURITY.md`.

## Scope of support

Best-effort support is provided for:

- current `main` branch
- latest tagged release

Older revisions may not receive fixes.
